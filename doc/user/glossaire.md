# Glossaire

Définitions des termes qui reviennent dans cette documentation, classés par ordre alphabétique.

## A

### ACF
Advanced Custom Fields. Un plugin WordPress qui permet d'ajouter des **champs personnalisés** à une page (au-delà du titre et du contenu classique). Dans ce thème, ACF sert par exemple à saisir le bloc bienvenue de la page d'accueil ou la photo de la page de contact.

### Alt (texte alternatif)
Description textuelle d'une image, lue à haute voix par un lecteur d'écran. Essentiel pour l'accessibilité. Dans Gutenberg, tu le remplis quand tu ajoutes une image (champ **Texte alternatif** dans le panneau latéral).

## B

### Bloc (Gutenberg)
Unité de contenu dans l'éditeur WordPress. Un paragraphe, un titre, une image, un bouton, un shortcode… chacun est un bloc qu'on peut ajouter, déplacer, supprimer indépendamment.

### Brouillon
État d'une page ou d'un article **non publié**. Visible uniquement par toi dans l'admin, invisible sur le site public.

## C

### Champ personnalisé
Donnée supplémentaire liée à une page, qui n'entre pas dans le titre ou le contenu classique. Exemple : la photo de Virginie sur la page de contact.

### Classe CSS
Étiquette qu'on colle à un bloc pour lui appliquer une mise en forme particulière. Les classes disponibles dans ce thème sont listées dans [03 · Mise en page avec des classes](03-mise-en-page-avec-des-classes.md).

### CPT (Custom Post Type)
Type de contenu personnalisé. Dans ce thème, les **Témoignages** et les **Événements** sont des CPT (en plus des types standards Page et Article).

### CTA (Call-To-Action)
Appel à l'action. Un bouton ou un lien qui invite le visiteur à faire quelque chose : « Me contacter », « Voir le calendrier »…

## G

### Gutenberg
Nom de l'éditeur visuel de WordPress (depuis la version 5.0). Il fonctionne par *blocs*. Synonyme : « éditeur de blocs ».

## L

### Lecteur d'écran
Logiciel qui lit le contenu d'une page à voix haute pour les personnes non-voyantes ou malvoyantes. Le site doit être pensé pour être compréhensible par ces logiciels (c'est l'objet de l'**accessibilité**).

## M

### Meta box (ou meta field)
Panneau supplémentaire dans l'éditeur Gutenberg, en dessous ou à côté du contenu principal, qui demande des informations spécifiques à la page (par exemple : la photo de contact, les mots-clés de la bienvenue, etc.).

## P

### Publier
Mettre une page ou un article en ligne, visible par tout le monde. Opposé à **Brouillon**.

## S

### Shortcode (code court)
Texte entre crochets qui, lorsqu'il est collé dans une page, génère automatiquement un contenu plus riche. Exemples du thème : `[kiyose_testimonials]`, `[kiyose_signets]`, `[kiyose_callout]`.

### Slug
Version simplifiée d'un titre, utilisée dans les URL. Le slug de la page « À propos » est `a-propos`. Tu peux le modifier dans le panneau latéral (onglet **Page** > **URL**).

## T

### Template (modèle de page)
Modèle de mise en page qu'on assigne à une page. Le template décide comment le contenu est présenté. Liste des templates disponibles : [02 · Choisir un template](02-choisir-un-template.md).

### Témoignage
Dans ce thème, les témoignages sont un type de contenu à part, géré dans le menu **Témoignages** de l'admin. Affichés via le shortcode `[kiyose_testimonials]`.

## W

### WordPress Admin
L'interface de gestion du site, accessible via `/wp-admin`. Uniquement visible par les personnes connectées avec les bons identifiants.
